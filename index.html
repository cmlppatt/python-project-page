Grade Calculator

Enter obtained grades and their weight to accurately find the current average

Python
Programming

March 2024

This program uses advanced list filtering techniques and lambda expressions to effeciently go through a given list of grades and given weight to calculate the current acheived grade. It is given users will input each grade following the syntax of [acheived score, max score, assessment type], where assessment type is one of quiz (Q), midterm (M) or final examination (FE). There can be any amount of assessments given at once per student.
